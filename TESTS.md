# Testing Notes

## Current Coverage

Vitest is used for validating:

* savings calculations
* recommendation logic
* annual projection calculations
* utility formatting behavior

## Why Testing Matters

The audit engine produces financial recommendations. Even simple logic errors could reduce trust in the generated savings estimates.

Automated testing helps ensure:

* stable calculations
* predictable outputs
* reliable recommendation behavior
