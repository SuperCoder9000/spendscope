# Pricing Data Notes

The application currently uses static pricing schemas for major AI tools.

Included providers:

* ChatGPT
* Claude
* Cursor
* GitHub Copilot
* Gemini

Pricing structures are manually defined to support:

* recommendation generation
* savings estimation
* annual projection calculations

Future improvements:

* live pricing APIs
* versioned pricing history
* enterprise pricing support
* usage-based pricing estimation
